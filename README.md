# simple_Wifi_Spammer

**Author:** lavashgovadina3  
**Platform:** Windows 7, 8, 8.1, 10, 11  

## Features
- 100+ SSID templates in 14 categories
- Auto-rotation of networks
- Random passwords or open networks
- Real-time statistics
- Configurable speed and limits

## SSID Categories
- Public WiFi, Coffee Shops, Fast Food, Hotels
- Telecom, Device Hotspots, Router Brands
- Government/Surveillance, Funny/Prank
- Educational, Airports, Tech Companies
- Gaming Networks, 5G Networks
